BetterLogs

Better Logs adds and an additional log line for all HTTP requests. Currently, it logs data in key=value pairs to allow for Splunk indexing.

format=html method=GET path=/ status=200  view_runtime=45.67 db_runtime=23.45 controller=rails/welcome action=index

Installation

Add this to your Gemfile

gem 'better_logs'
